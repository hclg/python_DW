# python 绘图库
---
> `Matplotlib` 是最著名的绘图库
### 1. 创建画布与创建子图

|函数名     |函数作用                   |
| -------- | ----------------------------------- |
|plt.figure|创建一个空白画布，可指定大小|
|figure.add_subplot| 创建并选中子图|
|plt.title|标题|
|plt.xlabel|x轴名称|
|plt.ylabel|y轴名称|
|plt.xlim|x轴的范围|
|plt.ylim|y轴的范围|
|plt.xticks|x轴刻度的数目与取值|
|plt.yticks|y轴刻度的数目与取值|
|plt.savafig|保存绘制图片|
|plt.show|本机显示图形|
